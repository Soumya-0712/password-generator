# Password Generator

A React app to generate secure, customizable passwords instantly.

## Features

- Adjustable password length (6–100 characters)
- Toggle numbers (0-9)
- Toggle special characters (!@#$%^&\*)
- One-click copy to clipboard

## Tech Stack

- React
- Tailwind CSS
- Vite
